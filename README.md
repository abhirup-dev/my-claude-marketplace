# My Claude Marketplace

A personal curated collection of Claude Code skills and plugins.

## Structure

```
my-claude-marketplace/
├── skills/          # Claude Code skill definitions (.md files)
├── mcp-servers/     # MCP server configurations
└── README.md
```

## Installing a Skill

```bash
# Install a skill from this marketplace
claude skill install <skill-name>

# Or manually copy the skill file
cp skills/<skill-name>.md ~/.claude/skills/
```

## Adding a Skill

1. Create a new `.md` file under `skills/`
2. Follow the skill format (see `skills/template.md`)
3. Commit and push
